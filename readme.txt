Post/Page specific custom css and javascript Plugin

== Description ==

A lightweight, easy to use custom css and javascript for WordPress. Ninja Custom Css Js use the metabox and methods, It will help you to write custom css and javascript for specific post or page. you can activate or deactivate css or js editor for page and post types. you can also choose At least PHP 5.3 is required.

It has some advanced features like:
 - Activate or deactivate css and javascript editor for page or different post type 
 - 25 theme for css editor
 - 25 theme for js editor

The code you write will be printed into footer.



*If you would like to implement any of the TODO please feel free to do and do a pull request*

And, finally, consider to contribute to this plugin [here](https://github.com/WpManageNinja/NinjaCustomCssJs).


== Installation ==
In the search field type “Ninja Custom Css Js” and click Search Plugins. Once you’ve found this plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.
